# Latihan-python
Hari ini aku mengilang kemabli pembelajaran python biar bisa lebih lancar berpikir dengan logika pemrograman.
Aku mulai dari menulis query print dan pesan. 
print() dalam Python adalah fungsi bawaan (built-in function) yang termasuk dalam kategori operasi Output (keluaran). Fungsinya adalah untuk menampilkan pesan atau data ke layar (konsol) agar bisa dilihat oleh pengguna atau untuk keperluan debugging. 
Pesan dalam Python biasanya merujuk pada string (untaian karakter), yang merupakan salah satu tipe data dasar di Python
Saat menulis print() kita bisa mengapit tulisan langsung yang akan menjadi keluaran dengan tanda pentik ganda ("...") maupun tanda petik tunggal ('...'). 
Saat menulis pesan kita gunakan simbol sama dengan (=). Jika ingin pesan tersebut menjadi keluaran, maka gunakan query print(pesan). Tidak perlu menggunakan kutip. Kenapa? Alasannya, saat kita mencetak nilai yang disimpan dalam sebuah variabel atau sebuah ekspresi, nggak perlu pakai kutip. Tanda kutip digunain untuk membedakan antara string literal dengan elemen pemrograman lainnya, seperti variabel atau kata kunci. 
String Literal: Tanda kutip (tunggal ' atau ganda ") digunakan untuk menandai rangkaian karakter sebagai sebuah string literal (nilai teks mentah yang ditafsirkan persis seperti yang dituliskan). Ketika Anda menulis print("Halo dunia"), program mengerti bahwa "Halo dunia" adalah teks yang harus ditampilkan.
Variabel: Saat Anda ingin mencetak nilai dari sebuah variabel, Anda tidak menggunakan tanda kutip. Alasannya adalah agar program mengenali nama tersebut sebagai referensi ke lokasi memori yang menyimpan nilai, bukan sebagai teks literal "nama variabel" itu sendiri.
Ekspresi dan Tipe Data Lain: Sama halnya dengan variabel, tipe data lain seperti angka (integer atau float), atau hasil dari suatu ekspresi matematika juga tidak memerlukan tanda kutip. Untuk ini akan kita pelajari di materi berikutnya.
